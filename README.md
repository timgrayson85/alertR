# alertR

<b>Get Started</b>

1. Download Node.js

https://nodejs.org/en/download/

2. Install socket.io

```
$ npm install socket.io
```

3. Install Express

```
$npm install --save express@4.15.2
```

4. To start the application:

```
$node app.js
```

5. Open a browser and point to <code>http://localhost:8080</code>
