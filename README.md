# alertR

<b>Get Started</b>

1. Download Node.js and npm package manager:

<code>https://nodejs.org/en/download/</code>

2. Install Dependencies:

```
$ npm install
```

3. Start the application:

```
$npm run dev
```

4. Let your colleagues know when something bad has happened:
